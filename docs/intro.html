Your First Game - TicTacToe

The Final Result​

You will learn:

  1. State management across frontend and backend
  2. Developing a turn based game locally
  3. Deploying your game to production

Pre-requesites​

Basic understanding of:

  1. JavaScript
  2. ReactJS
  3. Git

Installed:

  1. node

1. Fork the tictactoe repository​

Navigate to github.com/turnbasedgames/tictactoe and fork the repository:

Clone the forked repository and run the following command:

npm install
npm start

The developer server should start up. You should be able to start a simple game by adding players!

2. Deploying to production​

Navigate to urturn.app, and create an account if you haven't already. Click on create game and input the corresponding information for your game.
